== Description ==

Need SEO Backlinks? Let your visitors link back to your site with this simple but effective widget.

After plugin activation you will be automatically redirected to your widgets page so you can configure the Link to us widget in your sidebar.

That simple!

If you need help or you want to report a bug please visit http://seo.uk.net/link-to-us-widget/

== Installation ==

1. Upload the zip to 'plugins' directory or from WP Admin -> Plugins -> and search for Link to us.
1. Activate the plugin
1. You will be automatically redirected to your widgets page so you can configure the Link to us widget in your sidebar.

Link to us Widget does not have a settings section on Admin page.

If you need help with the plugin or you want to report a bug please refer to http://seo.uk.net/link-to-us-widget/
